Installation:
1. Copy Colref.dll to \RadASM\addins

2. Add text below: 
x=Colref.dll,1

to section named: [Addins]
in file RadASM.ini
*x is next free number

Usage:
Choose a color from the color dialog box or
pic a color from the screen. Insert color reference in code.
